◆ AVInterruptCallBackFun()

int playerMedia::AVInterruptCallBackFun ( void *  param)
static

ffmpeg的回调函数，跳出阻塞

根据成员变量interruptFlag 跳出打开视频流超时导致avformat_open_input()阻塞， 以及网络速度慢，长时间无法加载导致av_read_frame()阻塞， 函数阻塞如果不强制跳出会导致程序卡死。

返回
返回0继续执行，返回1退出阻塞
